## 🚀 Projects Overview

### Tripadvisor Clone
Located in `CSS/tripadvisor-clone/`.
A pixel-accurate clone of the Tripadvisor homepage built with **pure HTML and CSS** — no JavaScript.

- **Sticky Navbar** with logo, search bar, navigation links, and sign-in button.
- **Search Section** with category tabs (Search All, Hotels, Things to Do, Restaurants) using CSS radio-button logic to switch inputs and underline states.
- **Animated Banner** with a CSS-only image slider and a play/pause toggle using the checkbox hack.
- **Card Grids** for Explore, Adventures, Interest, and Iconic destinations sections.
- **Donation Section** with a split image/text layout.
- **Footer** with multi-column navigation, dropdowns, social icons, and a collapsible "Read more / Read less" section using `<details>` and `:has()`.
- **Fully Responsive** — mobile-first media queries adapt the navbar (hamburger + centered logo), search (scrollable tabs + stacked input/button), banner, donate, cards, and footer for small screens.

### Personal Portfolio

### Udemy Clone

### HTML Foundations

# **Tripadvisor Homepage Clone**

## **Overview**
This project is a high-fidelity frontend clone of the **Tripadvisor** homepage, completed as a **Week 2 assignment** for my Full-Stack Web Development certification. The primary focus of this project was to master complex UI structures, responsive design patterns, and "Docs as Code" workflows.

**Live Demo:** [https://madaswamy14.github.io/FSWD/CSS/tripadvisor-clone/index.html](https://madaswamy14.github.io/FSWD/CSS/tripadvisor-clone/index.html)

---

## **Technical Features**
*   **Adaptive Grid Layout:** Implemented a sophisticated grid system using **CSS Flexbox** that transitions seamlessly from 4-column desktop layouts to 2-column and 1-column mobile views.
*   **Responsive Media Logic:** Utilized precision media queries to ensure image banners and search components remain perfectly positioned across all device breakpoints.
*   **Interactive UI States:** Developed custom hover effects and transition states to mimic the real-world user interaction of the Tripadvisor platform.
*   **Modern CSS Architecture:** Focused on modular styling, using clean class naming conventions for better maintainability and scalability.

---

## **Tech Stack**
*   **HTML5:** Structured semantic markup for accessibility and SEO.
*   **CSS3:** Advanced Flexbox, Positioning, and Responsive Design.
*   **Git/GitHub:** Version control and automated deployment via GitHub Pages.
